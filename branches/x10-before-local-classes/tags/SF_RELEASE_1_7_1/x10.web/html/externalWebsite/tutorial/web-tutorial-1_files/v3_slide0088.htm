Example: RandomAccess (2/2)
(1)  Allocate and initialize table as a block-distributed array.
(2)  Allocate and initialize RanStarts with one random number seed
for each place.
(3)  Allocate a small immutable table that can be copied to all places.
(4)  Everywhere in parallel, repeatedly generate random table indices
and atomically read/modify/write table element.
