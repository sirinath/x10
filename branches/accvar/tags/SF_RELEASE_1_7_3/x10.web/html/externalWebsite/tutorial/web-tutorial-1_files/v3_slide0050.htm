Example: rooted exception model (future)
double div (final double divisor)
   future<double> f = future { return 42.0 / divisor; }
   double result;
   try {
     result = f.force();
   } catch (ArithmeticException e) {
     result = 0.0;
   }
   return result;
}
Exception is propagated when the future is forced.
