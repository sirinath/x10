Welcome to delhi.watson.ibm.com

As of 1/1/2005 the only project hosted on Delhi is X10.

X10 is an experimental language for concurrent and distributed programming being designed by IBM and University partners, initially under the aegis of the DARPA-sponsored PERCS project (PI Mootaz Elnozahy ). Jan Vitek (Purdue) is hosting the X10 CVS repository and the X10 Mantis bug tracking system at Purdue. (Both of these require user account, please contact me if you need them.)

This site is intended for developers of the X10 language, users, and others who may be interested in finding out more about X10. Please treat the information on this website as IBM Confidential, unless the document explicitly states otherwise.
