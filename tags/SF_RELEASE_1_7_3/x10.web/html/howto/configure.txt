HOWTO: Configure an X10 system that you checked out from CVS
------------------------------------------------------------

 - Make sure that the JAVA_HOME and the X10_PLATFORM environment 
   variables are set correctly (see x10.common/README)

From within eclipse:

 - Select x10.common/build.xml --> Run As --> ant script

 From the command line:
 
 - Make sure you have ant installed
 - cd x10.common
 - ant
