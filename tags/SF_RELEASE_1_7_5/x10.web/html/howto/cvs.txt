HOWTO: Access the CVS repository from the command line:
-------------------------------------------------------

What you need:

 - Windows or Linux installation with CVS.
 - an account in on quillback.cs.purdue.edu
 - your private ssh keys

Procedure:

- set the following environment variables:

   CVSROOT=:ext:<yourlogin>@quillback.cs.purdue.edu:/home/ibmcvs
   CVS_RSH=ssh
   CVSUMASK=770

Required:

  cvs checkout polyglot     // polyglot 1.3
  cvs checkout lpg          // parser generator
  cvs checkout pe           // performance explorer
  cvs checkout junit        // junit testing 
  cvs checkout x10.compiler // x10 translator
  cvs checkout x10.runtime  // x10 runtime
  cvs checkout x10.common   // scripts and config files common to runtime and compiler
  
  Optional:
  
  cvs checkout x10.lib        // x10 std library
  cvs checkout x10.test       // unit tests for compiler and runtime
  cvs checkout x10.vm         // vm interface
  cvs checkout x10.man        // x10 related documentations 
  cvs checkout x10.installer  // x10 installer package
  cvs checkout x10.design     // paper that influenced the x10 design
  cvs checkout x10web         // x10 web pages
  cvs checkout x10apps        // x10 JGF application in several variants
