Syntax extensions for distributions
Constant distributions
  region r = [0:N];
 dist d = r->here
    dist d = dist.factory.constant(r, here);
  dist d = 1000->here
            dist d = dist.factory.constant([0,1000],
here);
Distributions are implicitly converted to regions
for (point [i,j]: d) {...}
       for (point [i,j]: d.region) {...}
