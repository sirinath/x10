Memory safety
Runtime invariants
§ An object may only access
memory within its
representation, and other
objects it has a reference to.
– X10 supports no pointer
arithmetic.
– Array access is bounds-
checked dynamically (if
necessary).
§ No “ill mem ref”
– No object can have a
reference to an object who’s
memory has been freed.
– X10 uses garbage collection.
§ Every value read from a
location has been previously
written into the location.
– No uninitialized variables.
