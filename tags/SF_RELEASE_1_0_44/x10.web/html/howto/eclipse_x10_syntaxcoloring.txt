HOWTO: Enable Java syntax coloring for X10 in Eclipse:
------------------------------------------------------

What you need:

 - Eclipse 3.0 or later
 

Procedure:

- Choose "Window" -> "Preferences" -> 
         "Workbench" -> "File Associations" 
- Add File type: *.x10
- Associate "Java Editor" to this newly created entry.


Note:

X10 language features and keywords are not honored by this
mechanism.
